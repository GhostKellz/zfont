# Performance Evidence

ZFont does not currently have enough checked benchmark evidence to claim broad
speedups over HarfBuzz, ICU, FreeType, FontConfig, or Pango.

## Current Policy

- Treat performance claims as provisional until generated by repository benchmarks.
- Record runner metadata, Zig version, optimize mode, fixture fonts, and input text.
- Compare against specific versions of external libraries when publishing numbers.
- Keep benchmark thresholds informational until results are stable.

## Benchmark Areas

- font load and table parsing
- cmap lookup
- glyph lookup and metrics
- terminal cursor movement
- gcode-backed width/segmentation usage
- Arabic/Indic/BiDi shaping experiments
- emoji sequence handling
- fallback chain lookup

## Future Command Shape

```bash
zig build benchmark
zig build benchmark-compare
```

Only document those commands once they exist in `build.zig` and pass locally.
